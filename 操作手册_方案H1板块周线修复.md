# 方案 H.1：板块成分匹配 + 周线宽松模式

## 改动

1. 板块热度源调整：
   - 优先东方财富 `stock_board_industry_name_em()`；
   - 同花顺 `stock_board_industry_summary_ths()` 作为备用；
   - 原因：东方财富热度名称与 `stock_board_industry_cons_em(symbol=行业名)` 同源，成分股匹配更稳。

2. 周线过滤调整：
   - 默认调试模式：周线数据不足不剔除，先放行并标记 `weekly_status=skipped_not_enough`；
   - 严格模式：加 `--strict-weekly`，周线不足也剔除。

## 推荐测试

```bash
python run_scan.py --coverage
python run_scan.py --mode observe --workers 1
```

## 正式严格模式

```bash
python run_scan.py --mode tail_confirm --workers 1 --strict-sector --strict-weekly
```

## 观察重点

- 板块热度源是否显示 `eastmoney`；
- 强势板块成分是否不再为空；
- 周线报告里是否出现“周线不足但放行”；
- 是否进入日线二买扫描阶段。
